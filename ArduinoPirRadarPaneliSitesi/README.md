# Arduino PIR Radar Paneli Sitesi

Arduino PIR sensörü kullanarak hareket algılama sistemi için web arayüzü.

## Özellikler

- 🎯 Gerçek zamanlı hareket algılama
- 📊 Radar görseli ile görselleştirme
- 🎨 Responsive ve kullanıcı dostu arayüz
- ⚡ Hızlı ve hafif

## Dosyalar

- `index.html` - Ana HTML dosyası
- `style.css` - Stil dosyası
- `script.js` - JavaScript fonksiyonları

## Kurulum

1. Dosyaları indirin
2. `index.html` dosyasını tarayıcıda açın
3. Arduino'yu bağlayın ve kodu yükleyin

## Gereksinimler

- Arduino kartı
- PIR hareket sensörü
- Tarayıcı (Chrome, Firefox, Safari, Edge)
